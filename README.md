# VaultMate Password & Passkey Manager

VaultMate has evolved into a powerful, offline-first password manager with an integrated browser extension, bringing you cross-platform security with built-in Passkey interception!

## 📁 Project Structure

This repository is split into two distinct, interconnected components:

### 1. `Application/`
The core desktop password manager built with Python, CustomTkinter, and SQLite.
- **Offline First**: All data is stored locally in `vaultmate.db`. No cloud sync.
- **Bank-Level Encryption**: Passwords and Passkey Private Keys are encrypted via `cryptography.fernet` using PBKDF2 key derivation.
- **Unified GUI**: Manage your Web Passwords, App Passwords, and Passkeys seamlessly in one place.
- **Native Host**: Acts as a secure backend to process cryptographic WebAuthn requests directly on your machine.

👉 [View Application Documentation](Application/README.md)

### 2. `extension-chrome/` & `extension-firefox/`
A custom Manifest V3 browser extension that bridges your web browser with the VaultMate application. Due to strict browser differences, there are two versions of the extension included.
- **Cross-Browser**: Supports Chrome, Edge, Brave, and Firefox (including Flatpak sandboxes).
- **Passkey Interception**: Injects scripts natively into the MAIN world to transparently intercept WebAuthn Passkey creation and authentication.
- **Auto-Fill & Auto-Save**: Seamlessly injects credentials into login forms and automatically captures new passwords upon submission.

👉 [View Extension Documentation](extension/README.md)

## 🚀 Quick Start

1. **Install the Application**: Navigate to `Application/`, setup your python virtual environment, and install `requirements.txt`. Run `python main.py`.
2. **Load the Extension**: Navigate to `chrome://extensions/` or `about:debugging` in your browser, turn on Developer Mode, and Load Unpacked the `extension-chrome/` or `extension-firefox/` directory.
3. **Link Them**: Open the VaultMate GUI, go to the **Browser Integrations** tab, click "Install Extension Host", and paste the Extension ID generated by your browser.
